  1. use geom_jitter() to make another scatter plot
ggplot(data = morley, aes(Expt, Speed))+
  # jittered plot 
  geom_jitter(width = 0.1, height = 0, size = 1)+
  # titles 
  labs(title = "Speed vs experiment number",
       x = "Experiment")+
  theme(plot.title = element_text(hjust = 0.5))

The height and width in geom_jitter is adjusted. 1. Height : since the values in y-axis are continuous and the points should not be jittered, we set the height = 0 2. width : in the default setting, the point are jittered so widely that there is no distinction between experiment numbers. To avoid such a case, we reduce the width to 0.1.

  1. What is the advantage of a violin plot compared to a box plot? What is the disadvantage?

Advantage - Violin plot allows us to see the probability density at different experiment numbers.

Disadvantage - Boxplot shows a summary statistics (quantiles, for example).
